## 研究目的
Web会議における予備動作を用いた発話欲求推定手法の実現

## 研究課題
1. 予備動作に用いる特徴の検討
2. 予備動作の抽出区間の検討
3. 発話欲求推定に有効な特徴量の検討

## 発表実績

    - タイトル：Web会議の非発話区間における顔特徴量を用いた発話予測手法の検討
